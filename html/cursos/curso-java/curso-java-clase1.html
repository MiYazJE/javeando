Primeros Pasos

En esta clase entenderemos el funcionamiento de una programa básico en Java, para ello necesitamos entender que es una clase.

Una clase no es mas que el encapsulamiento de un programa, por ejemplo de un hola mundo. Para ejecutar un hola mundo en java deberemos crear una clase y crear una función main con un array de tipo String de argumentos que contendra los parámetros enviados por línea de comandos.
La estructura sería la siguiente: 

                        
/* Estructura basica de un programa en Java */

public class holaMundo {
    
    public static void main(String[] args) {
        System.out.println( "Hola Mundo" );
    }

}


Qué significa el array pasado como argumento en la método main?

Son argumentos que obligatoriamente cualquier método main en un programa Java debe contener. Los programas en java se pueden ejecutar en la cosola y permiten el paso de parametros de la siguiente forma.
Si en la linea de comandos ejecutamos el siguiente de programa con "java leerArgumentos 1 2 3 4 5": 

                        
/* Programa que imprime los parametros pasados por consola */

public class leerArgumentos {

    public static void main(String[] args) {

        System.out.println("Imprimiedo argumentos por consola");
        for (String n : args) {
            System.out.println( n );
        }

    }

}

La salida del programa sera el siguiente:

                        
Imprimiedo argumentos por consola
1
2
3
4
